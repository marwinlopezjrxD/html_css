    git add: Adds changes from the working directory to the staging area, preparing them for the next commit.
    Example (single file): git add my_file.txt
    Example (all files): git add . or git add --all

    git commit: Permanently records the staged changes in the repository's history with a descriptive message.
    Example: git commit -m "A brief description of the changes"
